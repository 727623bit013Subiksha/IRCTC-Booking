# 🚆 IRCTC Ticket Booking App Redesign

A UI/UX redesign project focused on improving the IRCTC ticket booking experience with better usability, simplified navigation, and a modern interface.

---

## 📖 Project Overview

This project reimagines the IRCTC ticket booking application by creating a cleaner and more user-friendly experience. The redesign focuses on making ticket booking easier, faster, and more intuitive for users.

---

## 🎯 Problem Statement

The existing IRCTC application has a complex booking flow and cluttered interface, making it difficult for users to book tickets efficiently. The goal of this project was to simplify the user journey and improve the overall booking experience.

---

## 💡 Solution

- Redesigned the ticket booking flow for better usability.
- Created wireframes and interactive prototypes using Figma.
- Improved navigation and visual hierarchy.
- Designed a clean and modern user interface.

---

## ✨ Key Features

- Simplified ticket booking process
- Easy train search and selection
- Improved navigation
- Clean and modern interface
- Responsive mobile design
- Interactive Figma prototype
